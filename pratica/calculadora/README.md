# 🧮 Calculadora interativa

Implemente (ou compare com [`calculadora.sh`](calculadora.sh)) uma calculadora com dois operandos e operações: soma, subtração, multiplicação e divisão. Pode usar menu para escolher a operação ou sair.

Enunciado detalhado e extras: [PLANO-EXERCICIOS.md](../PLANO-EXERCICIOS.md) (seção **15. Marco — Calculadora**, M1–M3).

## ▶️ ▶️ Executar

```bash
cd pratica/calculadora
bash calculadora.sh
```

*(O script pode tentar instalar `bc` via `apt` se não estiver disponível — ajuste conforme sua distribuição.)*
